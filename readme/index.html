发表博客：

  1. 打开当前目录的终端
  2. 终端执行hugo new post/File_Name.md新建文件
  3. 写文章
  4. 终端执行hugo server本地预览
  5. 修改格式，用 \_ replace _ 避免markdown与mathjax的语法冲突（暂时的解决办法）。看下面的注意2。
  6. 修改完毕
  7. 终端执行hugo在public文件夹生成相应的文件（此行为不可撤销，需谨慎）
  8. 运行update.sh把content和public上传到相应的github

本地站点丢失的恢复步骤：

  1. 下载hugo，新建站点
  2. 克隆主题even到文件夹themes
  3. pull content和public的内容
  4. hugo server预览
  5. 预览无误后同上述步骤7和8

注意1： hugo before update

注意2： _用\_代替 \{用\\{代替 *用\*代替 \\用\\\\代替
